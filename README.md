## SyncAdmin Legacy Build Artifacts
These JSON files contain Lua proto code responsible for executing SyncAdmin Legacy in your Roblox game. 

You can use these files to pick and choose which version of SyncAdmin Legacy you run within your game by following the steps here:
https://developer.polymatic.co/t/legacy-internal-development-process/82

### Warning!
Modifying these advanced settings can be harmful to the stability, performance and security of SyncAdmin. You should only modify these settings if you are sure of what you are doing or instructed to do so by a Polymatic Labs Engineer. Polymatic Labs cannot provide support for older versions of SyncAdmin.

In addition to the above, please understand the risks of modifying SyncAdmin internals. More information here:
https://developer.polymatic.co/t/syncadmin-modifying-source-code/92
